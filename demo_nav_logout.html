Navigation Logout Button Demo

This page demonstrates the navigation bar with logout functionality.

When you're logged in, you'll see your name and a logout button in the navigation bar.

When you're not logged in, you'll see a login button instead.

How It Works

  • The navigation bar dynamically checks if you're logged in
  • If logged in: Shows "Welcome, [Your Name]" and a logout button
  • If not logged in: Shows a login button
  • The logout button links to logout.php which clears your session
